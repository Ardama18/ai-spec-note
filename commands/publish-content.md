## note投稿専門原則（投稿・収益化エージェント向け）

### 🚨 最重要原則：分析OK、投稿前承認必須

**すべての投稿実行前にユーザー承認が必須**

理由：投稿取り消し不可能性と収益化戦略の確実性を確保するため

### 投稿実行プロセス

#### 実行フロー（必須手順）
1. **投稿準備完了確認** → 完成コンテンツの品質・最適化確認
2. **note最適化実装** → ハッシュタグ・マガジン・収益化設定
3. **投稿実行** → ユーザー承認後の実際の投稿
4. **初期フォローアップ** → 投稿後1-3時間の反応確認・対応

#### note投稿の前提条件
1. **コンテンツ品質確認完了** → AEO・GEO最適化済み
2. **プラットフォーム最適化実装** → note特化要件満足
3. **収益化戦略確定** → 価格設定・有料エリア設計完了
4. **ユーザー最終承認** → 投稿内容・戦略の明示的承認

#### 禁止事項（投稿時）
- **品質確認なしの投稿** → 基準未満は投稿禁止
- **承認なし投稿実行** → 承認待ちに移行
- **収益化設定未完了での投稿** → 収益機会損失防止

### 投稿時の行動制御（失敗防止）

#### 自動停止トリガー（必ず停止）
- **同時多数投稿検出**：即座停止、投稿戦略をユーザーに確認
  理由：noteアルゴリズムスパム判定回避のため
- **収益化設定エラー**: 収益化設定確認を強制
  理由：収益機会損失防止のため

#### エンゲージメント低下対処
1. 低エンゲージメント検出 → **一時停止**
2. 原因分析（タイミング・内容・設定の検証）
3. 改善計画提示
4. ユーザー承認後に対策実行

#### 投稿戦略の品質管理
**測定可能な品質基準**：
- **noteプラットフォーム最適化スコア**: 基準値以上
- **エンゲージメント設計完了**: 必須要素の実装確認
- **収益化戦略整合性**: 全体戦略との一貫性確認

## note投稿成功パターン

### 投稿準備段階
1. **完全最適化**: AEO・GEO・note特化最適化の完了
2. **戦略整合性**: 収益化戦略・エンゲージメント戦略の一貫性
3. **タイミング最適化**: noteアルゴリズム・読者行動パターンとの同期

### 投稿実行段階
1. **正確な設定**: ハッシュタグ・マガジン・価格設定の精度
2. **即座フォロー**: 投稿直後のエンゲージメント促進活動
3. **継続監視**: 初期反応の継続的モニタリング

### 効果最大化段階
1. **データ活用**: エンゲージメント・収益データの戦略反映
2. **継続改善**: A/Bテスト・最適化の継続実行
3. **長期戦略**: シリーズ・マガジン展開への発展

## リスク管理・品質保証

### 投稿リスク回避
1. **スパム判定回避**: 投稿頻度・内容の適切な管理
2. **エンゲージメント低下防止**: 読者ニーズ・トレンドとの整合性確保
3. **収益化トラブル回避**: 価格設定・価値提供の適切性確認

### 品質保証プロセス
1. **投稿前最終確認**: 全最適化要件・戦略要件の確認
2. **投稿後即座確認**: 表示・設定・初期反応の確認
3. **継続品質管理**: エンゲージメント・収益品質の継続監視

### エスカレーション基準
- **重大なエンゲージメント低下**: 即座にユーザー報告・戦略見直し
- **収益化設定問題**: 収益機会損失防止のための緊急対応
- **noteプラットフォーム問題**: プラットフォーム仕様変更等への対応
