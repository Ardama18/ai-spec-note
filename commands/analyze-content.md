## コンテンツ分析専門原則（分析・改善エージェント向け）

### 🚨 最重要原則：分析OK、改善実行前承認必須

**すべての戦略変更・改善実行前にユーザー承認が必須**

理由：分析結果の解釈誤り・過度な最適化による品質劣化を防ぐため

### 分析実行プロセス

#### 実行フロー（必須手順）
1. **データ収集・整理** → 複数ソースからの統合データ準備
2. **分析実行** → 定量・定性分析の実施
3. **インサイト抽出** → 戦略的示唆・改善ポイントの特定
4. **改善提案策定** → 具体的・実行可能な改善計画

#### 分析の前提条件
1. **十分なデータ蓄積** → 統計的信頼性確保のための最小データ量
2. **多角的視点** → AEO・GEO・エンゲージメント・収益の統合分析
3. **時系列比較** → トレンド・変化パターンの把握
4. **競合・ベンチマーク** → 相対的評価・市場ポジション確認

#### 禁止事項（分析時）
- **単一指標のみの判断** → 統合的評価の欠如防止
- **短期データのみでの判断** → トレンド誤認防止
- **承認なし戦略変更提案** → 承認待ちに移行

### 分析時の品質制御（誤認防止）

#### 分析品質確保
- **統計的信頼性**: 十分なサンプル数・期間での分析
- **因果関係vs相関**: 正確な因果関係の特定
- **季節性・外部要因**: 外部要因の適切な考慮
- **複数指標統合**: 単一指標依存の回避

#### 改善提案の妥当性確保
1. 改善提案の実現可能性検証
2. リソース・コスト・リスクの評価
3. 短期・中期・長期効果の予測
4. A/Bテスト・段階実装の計画

#### 戦略変更の慎重性確保
**測定可能な変更基準**：
- **重大な戦略変更**: 必ずユーザー承認・段階実装
- **継続的改善**: データに基づく小幅調整の継続
- **リスク評価**: 変更による潜在的リスクの事前評価

## 分析エージェント連携フロー

### データ収集・基礎分析フェーズ
1. **analytics-checker**: 包括的効果測定・基本ROI分析
2. **engagement-analyzer**: エンゲージメント詳細分析

### 市場・戦略分析フェーズ
1. **trend-researcher**: 市場動向・競合・機会分析
2. **series-coordinator**: シリーズ統合・長期戦略分析

### 統合・提案フェーズ
1. **全エージェント結果統合**: 総合的インサイト抽出
2. **改善提案策定**: 優先順位付き改善計画
3. **次期戦略設計**: データドリブン戦略調整

### エージェント呼び出し原則
- **段階的分析**: 基礎→詳細→統合の段階的深化
- **相互検証**: 複数エージェント分析結果の相互確認
- **統合的判断**: 全分析結果を統合した最終判断

## 分析活用・改善サイクル

### 短期改善（1-2週間）
1. **即効性施策**: 分析で特定された即座改善可能項目
2. **A/Bテスト**: 小規模テストでの効果検証
3. **微調整**: エンゲージメント・収益の微調整

### 中期戦略調整（1-3ヶ月）
1. **戦略修正**: 市場変化・効果分析に基づく戦略調整
2. **新規企画**: トレンド分析に基づく新企画立案
3. **最適化深化**: AI最適化・プラットフォーム最適化の深化

### 長期戦略進化（3ヶ月以上）
1. **全体戦略見直し**: 包括的効果分析に基づく戦略再設計
2. **新領域展開**: 市場機会分析に基づく新領域進出
3. **システム改善**: 制作・投稿・分析システムの継続改善

## データドリブン意思決定

### 分析結果の戦略反映プロセス
1. **データ検証**: 分析結果の信頼性・妥当性確認
2. **優先順位付け**: 効果・実現可能性・リソースでの優先順位
3. **段階実装**: リスク分散・効果検証のための段階実装
4. **継続監視**: 実装後の効果継続監視・調整

### 意思決定基準
- **定量的根拠**: 数値データに基づく客観的判断
- **統合的評価**: 複数指標・長期視点での総合評価
- **リスク・リターン**: 期待効果とリスクのバランス評価
- **戦略整合性**: 全体戦略・長期目標との整合性確認

### 継続改善メカニズム
1. **定期分析**: 定期的な効果測定・分析の実施
2. **学習・調整**: 分析結果からの学習・戦略調整
3. **予測・先読み**: トレンド分析による先行的戦略調整
4. **システム進化**: 分析・改善プロセス自体の継続改善
