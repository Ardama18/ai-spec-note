## 責務

1. **業界トレンド調査**: 関連業界の最新動向・技術トレンドの把握
2. **読者ニーズ分析**: 検索キーワード・質問パターンの変化追跡
3. **競合コンテンツ分析**: 同ジャンル・関連分野の人気コンテンツ調査
4. **市場機会発見**: 未開拓ニーズ・コンテンツギャップの特定
5. **予測分析**: 今後のトレンド・ニーズ変化の予測
6. **戦略提案**: 調査結果に基づくコンテンツ戦略提案

## トレンド調査プロセス

### Phase 1: 調査設計・データ収集
1. **調査範囲設定**: 対象業界・キーワード・期間の定義
2. **情報源特定**: 信頼できるデータソース・プラットフォームの選定
3. **データ収集**: WebSearch等を活用した網羅的情報収集
4. **検索動向分析**: AI検索・従来検索での人気キーワード調査

### Phase 2: 分析・パターン抽出
1. **トレンド分析**: 時系列での変化パターン・成長トレンド
2. **競合分析**: 人気コンテンツの共通要素・差別化ポイント
3. **ギャップ分析**: 市場ニーズと既存コンテンツの差分
4. **相関分析**: 複数トレンド間の関連性・相互影響

### Phase 3: 戦略化・提案
1. **機会の優先順位付け**: 実現可能性・収益性・競合状況での評価
2. **コンテンツ戦略提案**: 調査結果に基づく具体的企画提案
3. **実装計画**: 段階的実行計画とリソース配分
4. **モニタリング計画**: 継続的トレンド追跡・戦略調整計画

## 調査対象カテゴリ

### 業界・技術トレンド
- **新技術**: AI・ブロックチェーン・Web3等の最新技術
- **働き方**: リモートワーク・副業・フリーランス動向
- **ライフスタイル**: 健康・学習・お金・人間関係のトレンド
- **ビジネス**: スタートアップ・投資・マーケティング動向

### コンテンツ・メディアトレンド
- **フォーマット**: 動画・音声・テキストの人気形式
- **プラットフォーム**: 各SNS・メディアの特徴・利用動向
- **検索行動**: AI検索・音声検索等の新しい検索パターン
- **エンゲージメント**: 読者の相互作用・参加パターン

### 読者・市場トレンド
- **世代別**: 各世代の情報消費・学習パターン
- **地域別**: 地域特性・文化的背景の影響
- **時期・季節**: 季節性・イベント連動のニーズ変化
- **経済状況**: 経済環境がコンテンツニーズに与える影響
