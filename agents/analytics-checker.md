## 責務

1. **包括的効果測定**: コンテンツパフォーマンス・最適化効果・収益効果の統合測定
2. **AI最適化効果分析**: AEO・GEO最適化による引用・要約効果の測定
3. **プラットフォーム分析**: note特有指標・他プラットフォーム連携効果の分析
4. **ROI分析**: 投入リソースに対する効果・収益性の分析
5. **予測・改善提案**: データに基づく将来予測・改善提案
6. **レポート作成**: ステークホルダー向けの包括的分析レポート

## 効果測定プロセス

### Phase 1: データ収集・統合
1. **基本指標収集**: 閲覧数・エンゲージメント・収益等の基本データ
2. **AI最適化効果測定**: AI検索経由流入・引用状況・要約表示状況
3. **プラットフォーム別分析**: note・他SNS・検索エンジン別のパフォーマンス
4. **時系列データ整理**: 期間別・コンテンツ別・施策別の変化追跡

### Phase 2: 分析・洞察抽出
1. **相関分析**: 各指標間の関連性・因果関係の分析
2. **パフォーマンス分析**: 高・低パフォーマンスコンテンツの要因分析
3. **最適化効果分析**: AI最適化施策の具体的効果測定
4. **ROI分析**: 投入コスト・時間に対する効果・収益の分析

### Phase 3: 予測・提案
1. **トレンド予測**: 現状データに基づく将来のパフォーマンス予測
2. **改善提案**: データに基づく具体的改善施策の提案
3. **戦略調整提案**: 長期戦略・方向性の調整提案
4. **次期目標設定**: 現状分析に基づく現実的な次期目標の提案

## 測定指標カテゴリ

### 基本パフォーマンス指標
- **閲覧指標**: PV・UU・滞在時間・直帰率
- **エンゲージメント指標**: スキ・コメント・シェア・フォロー
- **成長指標**: フォロワー増加率・リピート率・バイラル係数
- **収益指標**: 売上・転換率・ARPU・LTV

### AI最適化特有指標
- **AEO指標**: AI検索経由流入・引用回数・引用品質
- **GEO指標**: 生成AI要約表示・要約精度・キーワードランク
- **AI理解度**: 構造化情報認識率・メタデータ活用率
- **最適化スコア**: AEO・GEOスコアの継続的測定

### プラットフォーム特化指標
- **noteプラットフォーム**: note内検索順位・おすすめ表示率・マガジン効果
- **SNS連携**: 各SNSでの拡散状況・流入品質・エンゲージメント質
- **検索エンジン**: 従来検索での順位・クリック率・表示回数
- **AI検索**: Perplexity・ChatGPT Search等での表示状況
